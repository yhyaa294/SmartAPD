# Panduan Generasi Aset Visual untuk SmartAPD

Dokumen ini berisi kumpulan prompt untuk menghasilkan aset visual yang konsisten untuk website SmartAPD.

## 1. Hero Section
**Prompt:**
```
Desain hero section untuk landing page website bernama SmartAPD.
Gaya: UI/UX desain, dark mode industrial yang modern dan profesional.
Latar Belakang: Sangat gelap, dengan sedikit gradien warna oranye dan hijau.
Teks Headline Utama: "Ciptakan Zero Accident. Otomatiskan Kepatuhan."
Teks Sub-Headline: "Sistem visi AI yang tidak hanya mendeteksi pelanggaran, tapi juga menganalisis risiko untuk mencegah kecelakaan sebelum terjadi."
Tombol: Dua tombol - "Hubungi Sales (WA)" (oranye) dan "Lihat Demo" (outline putih).
```

**Catatan:**
- Gunakan resolusi 1920x1080px
- Simpan sebagai `hero-section.jpg`

## 2. Siklus Keselamatan Cerdas
**Prompt:**
```
Desain UI/UX untuk bagian "Siklus Keselamatan Cerdas" sebuah website teknologi.
Gaya: Tema dark mode industrial, konsisten dengan hero section.
Elemen: 4 kartu dalam satu baris horizontal dengan ikon dan teks:
1. "DETEKSI" dengan ikon kamera AI
2. "ANALISIS" dengan ikon grafik/otak
3. "AKSI" dengan ikon lonceng/notifikasi
4. "INTELIJEN" dengan ikon peta risiko/dokumen
Tampilkan panah atau garis yang menghubungkan keempat kartu.
```

**Catatan:**
- Gunakan resolusi 1920x800px
- Simpan sebagai `safety-cycle.jpg`

## 3. Alasan Memilih Kami
**Prompt:**
```
Desain bagian website "Mengapa Memilih Kami" dengan tema:
Judul: "Investasi K3, Bukan Sekadar Biaya"
Gaya: Dark mode profesional, palet warna oranye, hijau, dan hitam.
Elemen: 4 kartu (grid 2x2) dengan ikon dan judul:
1. "Turunkan Biaya Insiden" (ikon uang)
2. "Tingkatkan Produktivitas K3" (ikon grafik naik)
3. "Pastikan Kepatuhan & Audit" (ikon centang/dokumen)
4. "Bangun Budaya Keselamatan" (ikon tim/shield)
```

**Catatan:**
- Gunakan resolusi 1920x1000px
- Simpan sebagai `benefits.jpg`

## 4. Banner Pasalku.ai
**Prompt:**
```
Desain banner penawaran khusus untuk landing page website.
Gaya: Eksklusif, dark mode, dan eye-catching.
Konten:
- Teks: "Penawaran Khusus untuk 10 Mitra Peluncuran Pertama"
- Visual: Logo SmartAPD + Logo Pasalku.ai berdampingan
- Teks Promo: "GRATIS 3 Bulan Akun Premium Pasalku.ai (Senilai 99k)"
- Tombol: "Daftar Sekarang" dengan gaya menonjol
```

**Catatan:**
- Gunakan resolusi 1920x600px
- Simpan sebagai `partnership-banner.jpg`

## 5. Ikon dan Elemen Pendukung
**Prompt untuk Ikon:**
```
Buat ikon vektor dengan gaya line art modern untuk:
1. Deteksi APD (kamera dengan centang)
2. Analisis Risiko (grafik garis naik dengan tanda seru)
3. Notifikasi (lonceng dengan notif)
4. Laporan (dokumen dengan grafik)
5. Integrasi API (rantai/penghubung)
```

**Catatan:**
- Format SVG atau PNG transparan
- Ukuran 64x64px dan 128x128px
- Warna utama: #F97316 (oranye) dan #10B981 (hijau)

## Petunjuk Penggunaan
1. Gunakan tools seperti:
   - Winsurf/Copilot Designer
   - Figma
   - Adobe XD
   - Canva

2. Pastikan konsistensi:
   - Warna utama: #0F172A (dark blue), #F97316 (oranye), #10B981 (hijau)
   - Font: Gunakan font sans-serif modern (contoh: Inter, Poppins)
   - Spacing: Gunakan kelipatan 4px (4, 8, 12, 16, 24, 32, 48, 64, 96, 128, 192, 256, 384, 512, 640, 768)

3. Nama file:
   - Gunakan format kebab-case
   - Tambahkan versi jika perlu (contoh: `hero-section-v2.jpg`)

4. Simpan semua aset di folder `public/images/ai-generations/`
